이기종 산업 인프라 연동 (해상물류 및 제조)

다국적 파트너(이집트 등) 협업 및 외부 인터넷 접속이 제한되는 폐쇄망 환경의 정부과제 인프라를 설계했습니다. 이기종 장비 간 실시간 텔레메트리 데이터를 연동하기 위한 TCP/IP 통신 규격 및 DB 이중화 검토를 수행했습니다.

Root Cause Analysis

Incident Context Analysis & Resolution
DB Connection Pool Leak 특정 로직 내 트랜잭션 미해제로 인한 활성 커넥션 누수 식별. Pool 반환 강제화 및 커넥션 타임아웃 튜닝을 통해 장애 격리 및 MTTR 단축.
Concurrent Update Collision 이벤트 보상 시 동시 다발적 API 호출로 동일 Row 데드락 발생. 비즈니스 로직 쿼리 순서 정렬 및 락(Pessimistic Lock) 스코프 최소화 적용.
CI/CD Pipeline Breakdown Ubuntu 서버 환경에서 배포 프로세스 관리 및 SSH 인증 권한 충돌 이슈 디버깅. 무중단 서비스 유지를 위해 배포 쉘 스크립트 리팩토링.
