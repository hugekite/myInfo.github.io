Architecture Insights

1. 신소리아 메타버스: 재화 시스템 정합성 보장

가상 경제 시스템의 재화 및 아이템 지급 로직을 설계했습니다. 보상 지급 시 우편함, 유저 소유권, 인벤토리 테이블이 갱신되는 과정에서 네트워크 장애 발생 시 데이터가 오염되는 것을 방지하기 위해 트랜잭션 롤백 구조를 적용했습니다.

2. VOTMS: AI 인재평가 실시간 스트리밍 최적화

기존 REST API 방식의 오디오 처리 파이프라인이 가진 순차적 응답 지연 한계를 극복하기 위해, 시스템 아키텍처를 WebSocket 기반의 실시간 양방향 스트리밍으로 전면 전환했습니다.

WebSocket Voice Orchestration

> Result: 오디오 수신 및 분석 지연 시간(Latency) 최소화 달성

3. 실시간 일일 랭킹 산정 최적화

전체 누적 랭킹이 아닌, 당일 발생한 활성 데이터를 기준으로 하는 실시간 일일 랭킹 시스템을 구축했습니다. RDBMS 조회 병목을 해소하기 위해 Redis를 도입하여 읽기/쓰기 성능을 극대화했습니다.

Redis Sorted Set TTL Key Rotation

4. 이기종 산업 인프라 연동 (해상물류)

외부 인터넷 접속이 제한되는 환경 및 다국적 비즈니스 파트너(이집트 등)와의 협업 프로젝트입니다. 정부과제 수준의 보안 요건과 이기종 장비 간 실시간 텔레메트리 데이터 연동을 위한 TCP/IP 통신 규격 설계 및 DB 이중화를 검토했습니다.
